1.2.3 Inputting Mixed Numbers

Mixed numbers are a combination of a whole number and a (proper) fraction. For example, we write the value \(2+\frac34\) more simply as a mixed number \[2\,\frac34\] by juxtaposition. This is a common notation in Imperial length measurement. Mixed numbers are equivalent to improper fractions which have numerators that are larger than the denominator. For example, the mixed number \(2+\frac34\) is equivalent to the improper fraction \(\frac{11}{4}\).

To input a mixed number:

  • input the whole number, then

  • press the <kbd class=“kbd-functions”>a<sup>b</sup>/<sub>c</sub></kbd> key, then

  • input the numerator’s value, then

  • press the <kbd class=“kbd-functions”>a<sup>b</sup>/<sub>c</sub></kbd> key, then

  • input the denominator’s value.

The mixed number will be displayed on the calculator.

[[style: callout]]

Example

To input the mixed number \(3\,\frac45\) into the calculator press:

3 <kbd class=“kbd-functions”>a<sup>b</sup>/<sub>c</sub></kbd> 4 <kbd class=“kbd-functions”>a<sup>b</sup>/<sub>c</sub></kbd> 5

The calculator will display <strong>3 _ 4 <span style="vertical-align: bottom;">&#x2A3C;</span> 5</strong> (the mixed number \(3\,\frac45\) ).

[[/style]]

1.2.9 Decimals with the TI-30Xa Calculator

Decimal notation represents an extension of the usual base-10 place-value number system to include values between whole numbers. For example, the decimal 15.4 represents a value bigger than fifteen but less than sixteen; similarly, the decimal representation of the number \(\sqrt{2}=1.41421\ldots\) (the length of the diagonal in a square of side length one) represents a value between one and two.

The TI-30Xa calculator has a limited display, so it can only display terminating decimals (i.e., decimals with only a certain number of digits); values for non-terminating decimals like \(\sqrt{2}=1.41421\ldots\) are displayed to a maximum of ten digits. As we set the calculator to display at most two decimal placeholders, this is not a significant issue; however, there are situations where the limitation of the display may limit the features of the calculator.

For example, the calculator can convert decimals into fractions; however, the fraction may be “too big” to display, so no conversion is done. For most examples in this course, the calculator should be able to convert decimals to an equivalent fraction.

Decimal values can be entered by pressing the <kbd class="kbd-numbers">.</kbd> key at the appropriate time.

[[style:instruction]]

Example

To enter 19.5 press:

  1. 9 <kbd class="kbd-numbers">.</kbd> 5

[[/style]]

1.2.12 Percentages with the TI-30Xa Calculator

By definition, percentages compare a numerical value to the number one hundred (“per hundred”). Percentages are another convenient representation of numbers and are a common means of comparing two quantities. For example, we may compare deductions to gross pay to get a sense of how much is being deducted from a paycheck relative to what we earned.

Depending on the situation, it may be convenient to work with percentages, decimals, or fractions, so conversion between these forms is beneficial. In particular, percentages are often changed to their decimal equivalents. The TI-30Xa calculator has a built-in feature to do this conversion, which makes calculations such as percentage changes quick and easy.

[[style:standout]]

Note

The calculator has no built-in function for converting from decimal to percentage.

[[/style]]

To convert a percentage to a decimal:

  • input the percentage value, then

  • press <kbd class="kbd-2nd">2nd</kbd>, then

  • press 2

[[style:standout]]

Example

To convert 56% to a decimal press:

5 6 <kbd class="kbd-2nd">2nd</kbd> 2

The calculator will display 0.56.

[[/style]]

1.3.0: Use Order of Operations to Simplify Expressions

Whenever multiple distinct operations like addition, subtraction, multiplication or division are required for a computation you must perform those operations in a certain order. The proper order so that computations are performed correctly is commonly known as the order of operations. This ordering requires you to:

  1. complete operations contained by brackets, then

  2. evaluate any exponents, then

  3. perform multiplication or division, then finally

  4. perform addition or subtraction.

Brackets (or parentheses) are used to isolate certain components so, naturally, these components are computed first.

In this section you will consider how to use the bracket (parenthesis) features of the TI-30Xa to group calculations so that order of operations is performed properly. You will also consider how to calculate exponents using the calculator.

[[style:standout]]

Note

The calculator can follow the order of operations for relatively simple calculations; however, it is recommended that you utilize brackets to ensure that your computations are properly interpreted by the calculator.

[[/style]]

1.4.0: Use the Calculator Memory

When performing calculations involving multiple steps it becomes easy to lose track of intermediate values. Additionally, if those values are not terminating decimals like 13.5 or 9.125 but represent irrational numbers like \(\pi=3.1415926…\) you may be tempted to round them to a more convenient value like \(\pi\approx 3.14\). This is not recommended as rounding skews from the true value (potentially by a significant amount) and may require careful record-keeping on your part.

To avoid introducing unnecessary rounding error and to save you from record-keeping, the TI-30Xa has three built-in memory registers that may be used to store numerical values.

The calculator will display M2 to indicate that a value has been stored in memory slot two.

1.4.3 Erasing a Value Stored in Memory

To erase a single memory slot:

  • press <kbd class="kbd-numbers">0</kbd>, then

  • press either <kbd class="kbd-numbers">1</kbd>, <kbd class="kbd-numbers">2</kbd> or <kbd class="kbd-numbers">3</kbd> corresponding to the memory slot to be erased.

[[style:standout]]

Example

To erase the value 3.14 that was stored into memory slot two:

  • press <kbd class="kbd-numbers">0</kbd>, then

  • press <kbd class="kbd-functions">STO</kbd> , then

  • press <kbd class="kbd-numbers">2</kbd>

The calculator will no longer display M2; the value in memory slot two was erased.

[[/style]]

[[style:standout]]

Note

All memory slots of the calculator are reset if the <kbd class="kbd-functions">OFF</kbd> key is pressed!

Pressing the <kbd class="kbd-functions">ON/C</kbd> key does not erase values stored into memory.
